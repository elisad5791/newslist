# Использование Redis в PHP-приложении для кеширования, хранения сессий и быстрых данных

### Проект курса Базы данных

## Приложение Список новостей

### Решенные задачи

1. Кеширование
2. Хранение сессий пользователей
3. Хранение других данных - просмотры, лайки
4. Очередь для асинхронной обработки задач
5. Контроль количества запросов пользователей
6. Синхронизация данных основного хранилища (MySQL) и быстрого хранилища (Redis)
7. Хранение и использование геоданных пользователей

### Использованные технологии

1. **Redis 7**
2. **PHP 8.3**
3. **Slim 4** 
4. **MySQL 8.0** 
5. **Docker** 

### Запуск проекта

1. склонировать проект

```bash
git clone git@github.com:elisad5791/newslist.git
```

2. перейти в директорию проекта

```bash
cd newslist
```

3. установить зависимости

```bash
composer install
```

4. поднять в docker контейнеры с MySQL, Redis

```bash
docker compose up -d
```

5. запустить встроенный сервер

```bash
cd public
php -S localhost:8888
```

6. открыть проект в браузере http://localhost:8888
